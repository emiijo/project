CSS

What is it?

  • CSS stands for Cascading Style Sheets.
  • CSS is used to describe how HTML elements are to be displayed on screen, paper, or in other media.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once.
  • External stylesheets are stored in CSS files.


Syntax to write CSS:

Selectors: The element(s) on which style should be applied.

Property and it's value: This is the actual style to be applied to the element(s).

  • When tags like 'font', and color attributes were added to the HTML 3.2 specification, it started a nightmare for web developers. Development of large websites, where fonts and color information were added to every single page, became a long and expensive process.
  • To solve this problem, the World Wide Web Consortium (W3C) created CSS.
  • CSS removed the style formatting from the HTML document!


CSS - problem solver

  • HTML was NEVER intended to contain tags for formatting a web page.
  • HTML was created to describe the content of a web page, like:

This is a heading

This is a paragraph.



3 places to write CSS: (in angle brackets)

A) Inline Styles: element style="property:value;. - Here you add the style directly in the HTML document, inside the element tag.


B) In the style element: - Where you add the style you want to add, in the header in order to add it to a group of selectors.


C) In a dedicated file (stylesheet.css) and refer via the "link" element:

Here you create your own CSS file, and link it to the HTML document, so that you can do the CSS styling separately and more organized.
