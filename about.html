HTML

HyperText Markup Language

  • HTML stands for HyperText Markup Language.

  • HyperText Markup Language was developed in the late 1980s and early 1990s to describe web pages.
  • We use HTML to label and accommodate content on web pages.
  • And the first document while making a web page (HTML) for coding is always called "index.html".
  • Let's go through the basic form and syntax for elements, attributes and entities.

Elements and building a HTML document:

  • In order to build a HTML document, it requires assembling a sequence of elements.
  • In HTML, the characters surrounded by angle brackets are called "tags".
  • Tags and the content between (and inside) elements are also called elements.
  • Elements may contain name/value pairs, called attributes. If an element has attributes, they always go in the the start-tag.
  • Elements are generally organized in a hierarchal way; with elements nested within elements).
